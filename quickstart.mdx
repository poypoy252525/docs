## Step 1: Install Hianime

### Using npm

```bash
npm install hianime
```

## Step 2: Initialize Hianime

### Usage

```ts
import Hianime from "hianime";

async function main() {
  const hianime = new Hianime();

  const result = await hianime.getTopAiring();

  console.log(JSON.stringify(result, null, 2));
}

main();
```
